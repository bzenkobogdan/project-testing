# How install

1. Download mongodb: https://www.mongodb.org/

2. Download nodejs: https://nodejs.org/

3. Run in console with project:

 1 - npm install

 2 - bower install

 3 - gulp

 4 - node server.js

4. Open browser with url: 'http://localhost:1337'
